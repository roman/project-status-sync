## Tests Added

### CursorIntegration group (4 unit tests)

1. **re-filter with saved cursor returns empty** — filters with cursor=0, re-filters with returned cursor, verifies empty
2. **cursor persists across write/read and prevents re-extraction** — full file I/O flow: read → filter → write → read → re-filter → verify empty
3. **multi-session cursors are independent** — two sessions with separate cursors, verify A's cursor doesn't affect B
4. **appended transcript yields only new content on re-filter** — simulates continued session (transcript grows), verifies only new messages extracted

### Property test (in filterProps)
