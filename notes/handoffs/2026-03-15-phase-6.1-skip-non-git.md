# Handoff: Phase 6.1 — Skip Non-Git Projects

**Date**: 2026-03-15
**Role**: Implementer

## What Was Done

Changed `identifyProject` to return `Maybe Project` instead of `Project`,
removing the `directoryFallback` that fabricated project identity from bare
directory names. Sessions from non-git directories (e.g. `~/`) now skip
processing entirely — no LLM calls, no events, no handoffs.

Key change: `identifyProject` is checked **before** transcript filtering
and LLM extraction, so non-git sessions cost zero LLM tokens (code-critic
caught the initial placement after extraction and it was moved up).

Files changed:
- `src/CCS/Project.hs`: return type `m Project` → `m (Maybe Project)`,
  `directoryFallback` deleted, unused `takeFileName` import removed
- `src/CCS/Process.hs`: `processSession` checks `identifyProject` first,
  skips with `logInfo` on `Nothing`
