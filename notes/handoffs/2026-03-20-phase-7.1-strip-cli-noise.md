# Handoff: Phase 7.1 — Strip CLI Noise from LLM Output

**Date**: 2026-03-20
**Role**: Implementer

## What Was Done

The `claude` CLI wrapper emits npm/pnpm lifecycle messages on stdout
before the LLM's actual response. These polluted all output files
(EVENTS.jsonl, handoffs, progress.log, STATUS.md).

`stripCLINoise` drops leading lines until the first line starting with
a recognized output marker: `[` (extraction tags), `#` (markdown), `{`
(JSON), a digit (progress dates), or `TOPIC:`. Applied centrally in
`runLLMPrompt` before `stripCodeFences`, so all four output paths
benefit.

Files changed:
- `src/CCS/Process.hs`: `stripCLINoise` function + `RIO.Char` import +
  wired into `runLLMPrompt` output pipeline
- `test/Main.hs`: 9 test cases (npm noise, markdown, JSON, progress,
  TOPIC, clean passthrough, no markers fallback, empty input)
